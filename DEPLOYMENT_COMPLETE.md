## 📋 NEXT STEPS (Required Setup)

### 1️⃣ Enable GitHub Pages (2 minutes)

1. Go to: https://github.com/aispiringcoder4302/indian-price-tracker/settings/pages
2. Under **Source**:
   - Branch: `main`
   - Folder: `/web`
3. Click **Save**
4. Wait 2 minutes
5. Your dashboard will be live at:
   **https://aispiringcoder4302.github.io/indian-price-tracker/**

### 2️⃣ Add GitHub Secrets (5 minutes)

Go to: https://github.com/aispiringcoder4302/indian-price-tracker/settings/secrets/actions

**Required Secrets** (for basic functionality):

| Secret Name | Value | How to Get |
|------------|-------|------------|
| `SENDER_EMAIL` | your-email@gmail.com | Your Gmail |
| `SENDER_PASSWORD` | app-password-here | [Generate here](https://myaccount.google.com/apppasswords) |
| `TELEGRAM_BOT_TOKEN` | 123:ABCxyz... | Talk to [@BotFather](https://t.me/botfather) |
| `TELEGRAM_CHAT_IDS` | 123456789 | Talk to [@userinfobot](https://t.me/userinfobot) |

**Optional Secrets** (for Google Sheets):

| Secret Name | Value |
|------------|-------|
| `SHEETS_CREDENTIALS` | Full JSON from service account |
| `SHEETS_ENABLED` | true |

### 3️⃣ Test Run (1 minute)

1. Go to: https://github.com/aispiringcoder4302/indian-price-tracker/actions
2. Click **"Indian Price Tracker - 24x7"** workflow
3. Click **"Run workflow"** → **"Run workflow"**
4. Wait ~1 minute for completion
5. Check Telegram/Email for test notifications!
